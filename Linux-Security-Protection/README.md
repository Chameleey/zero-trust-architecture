# 🔐 Linux Security Protection & Inspection

## 📄 Research Title
**Researches on the Security Protection and Inspection Method for Confidential Documents Based on Linux Operating System**

## 👩‍💻 My Contribution
- Core security principles in Linux
- User management and access control
- Conclusion and analysis

## 📌 Overview
This research focuses on security protection techniques and inspection methods in Linux operating systems. It highlights how confidential data can be secured using system-level controls, monitoring tools, and best practices.

## 🔍 Key Topics

### 🔐 Core Security Principles
- User management (useradd, usermod, userdel)
- File permissions (chmod, chown)
- Access control and sudo privileges

### 📂 Virtual File System (VFS)
- Secure file system abstraction
- Encrypted file systems
- File integrity monitoring (AIDE, Tripwire)

### 🛠️ Security Inspection Tools
- OpenVAS, Nessus (vulnerability scanners)
- Wireshark, tcpdump (packet sniffers)
- John the Ripper, Hashcat (password tools)
- Logwatch, Graylog (log analysis)
- ELK Stack, Splunk (SIEM)

### 🛡️ Security Practices
- File encryption
- Log monitoring
- System updates
- Firewalls
- Backups
